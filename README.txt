一、项目概述
================================================================================

本项目是一个面向生物医药领域的全栈数据分析平台，旨在为生物医学研究人员
提供从数据获取、清洗、统计分析到可视化报告生成的一站式智能分析解决方案。

平台整合了多种生物医学公共数据库接口、自动化统计分析引擎、AI智能分析
助手以及专业的报告生成系统，大幅降低生物医学数据分析的技术门槛。

【核心亮点】
  - 支持6大生物医学公共数据库一键获取数据
  - 智能自动分析引擎，根据数据特征自动选择最优统计方法
  - 8种回归分析方法自动推荐与模型比较
  - AI智能分析结论生成与对话式顾问
  - 一键生成专业Word/PDF分析报告
  - 生物医药主题配色，DNA双螺旋背景设计

三、项目创新点
================================================================================

【创新点一：多源生物医学数据统一接入与智能解析】
  传统痛点：生物医学数据分散在多个数据库，格式各异，获取和解析困难。
  创新方案：
    - 统一整合NCBI GEO、TCGA/GDC、UCI ML、WHO GHO、EBI ArrayExpress、
      Kaggle等6大生物医学数据库
    - 自动识别和解析不同格式的数据文件（CSV、Excel、GEO矩阵等）
    - 智能字段类型识别（数值型、分类型、日期型）
    - 一键获取，无需手动下载和格式转换
  技术价值：大幅降低数据获取门槛，提高研究效率。

【创新点二：智能分析方法推荐引擎】
  传统痛点：研究人员需要根据数据特征手动选择统计方法，容易出错。
  创新方案：
    - 自动进行数据特征分析（分布、正态性、方差齐性等）
    - 基于数据特征智能推荐最优统计方法
    - 正态性检验 -> 自动选择参数检验或非参数检验
    - 方差齐性检验 -> 自动选择等方差或异方差检验方法
    - 提供方法选择依据和解释
  技术价值：降低统计学专业知识门槛，确保分析方法选择的科学性。

【创新点三：多模型自动比较与最优选择】
  传统痛点：回归分析中模型选择依赖经验，难以全面比较。
  创新方案：
    - 同时训练OLS、稳健回归、岭回归等多种模型
    - 基于R²、RMSE、AIC、BIC等多维度指标自动比较
    - 智能推荐最优模型并提供选择理由
    - 自动检测多重共线性并推荐处理方法
  技术价值：确保模型选择的客观性和科学性，提高分析结果可靠性。

【创新点四：AI驱动的分析结论自动生成】
  传统痛点：统计分析结果需要人工解读和撰写结论，耗时且易出错。
  创新方案：
    - 基于大语言模型（豆包大模型）自动生成专业分析结论
    - 涵盖统计意义和临床/生物学意义双重解读
    - 支持中英文双语输出
    - 对话式AI顾问，支持多轮交互式问答
  技术价值：将AI技术深度融入数据分析流程，实现智能化分析。

【创新点五：全维度数据质量诊断与改进建议】
  传统痛点：数据质量问题往往被忽视，影响分析结果可靠性。
  创新方案：
    - 多维度数据质量检查（缺失值、异常值、重复值、分布特征等）
    - 问题严重程度分级（高/中/低）
    - 针对性改进建议（数据变换、非参数检验、样本量扩充等）
    - 清洗过程完整记录和可追溯
  技术价值：确保数据分析质量，提高研究结果的可信度。

【创新点六：一键式专业报告生成】
  传统痛点：研究报告撰写耗时，格式不规范，图表插入繁琐。
  创新方案：
    - 一键生成专业学术格式Word报告
    - 自动包含完整分析流程和方法说明
    - 图表自动插入、编号和标注
    - 支持多语言（中文/英文）和自定义模板
    - 报告内容结构化，便于后续修改和投稿
  技术价值：大幅缩短报告撰写时间，提高报告质量和规范性。

【创新点七：8种回归分析方法全覆盖】
  传统痛点：多数工具只支持基础回归方法，难以应对复杂数据场景。
  创新方案：
    - 多元线性回归（OLS）- 基础分析
    - 岭回归（Ridge）- 处理多重共线性
    - Lasso回归 - 特征筛选和变量压缩
    - 弹性网回归（Elastic Net）- 共线性+特征筛选
    - 稳健回归（Huber）- 抗异常值干扰
    - 分位数回归（Quantile）- 处理异方差
    - 虚拟变量回归 - 自动处理分类变量
    - 贝叶斯回归 - 小样本数据分析
  技术价值：覆盖各种数据场景，提供全面的回归分析能力。

【创新点八：交互式与静态可视化双引擎】
  传统痛点：单一可视化方式难以满足多样化展示需求。
  创新方案：
    - 静态图表：直方图、箱线图、小提琴图、柱状图、散点图、热力图
    - 交互式图表：支持缩放、平移、悬停查看的Plotly图表
    - 图表自动编号和标注
    - 图表一键导出
  技术价值：提供多样化的数据可视化方案，满足不同场景需求。

五、功能模块详解
================================================================================

5.1 数据自动获取模块
--------------------------------------------------------------------------------
支持从以下6大生物医学数据库自动获取数据：

  (1) NCBI GEO (Gene Expression Omnibus)
      - 全球最大的基因表达数据库
      - 支持GSE编号查询下载
      - 自动解析GEO系列矩阵文件

  (2) TCGA/GDC (The Cancer Genome Atlas)
      - 癌症基因组图谱数据库
      - 支持多种癌症类型数据查询
      - 自动获取临床与分子数据

  (3) UCI Machine Learning Repository
      - 经典学术数据集仓库
      - 支持帕金森病、乳腺癌等数据集
      - 自动解析UCI格式数据

  (4) WHO GHO (Global Health Observatory)
      - 世界卫生组织全球健康数据
      - 支持公共卫生指标查询
      - 自动获取全球健康统计数据

  (5) EBI ArrayExpress
      - 欧洲生物信息学研究所基因表达库
      - 支持ArrayExpress编号查询

  (6) Kaggle公开数据集
      - 通过GitHub镜像获取高质量数据集
      - 支持糖尿病、心脏病等经典数据集

5.2 数据手动获取模块
--------------------------------------------------------------------------------
支持两种本地数据导入方式：

  (1) 拖拽上传
      - 拖拽CSV/Excel文件到上传区域
      - 实时显示文件名和大小
      - 自动上传至服务器

  (2) 选择本地文件
      - 点击按钮选择本地文件
      - 支持CSV、Excel (.xlsx/.xls)格式
      - 自动识别列类型（数值/分类/日期）

5.3 智能分析引擎
--------------------------------------------------------------------------------
平台根据数据特征自动选择最优分析方法：

  【描述性统计】
      - 均值、标准差、中位数、最小值、最大值
      - 偏度、峰度、缺失值统计
      - 数据分布特征描述

  【正态性检验】
      - Shapiro-Wilk检验
      - 自动判断数据是否服从正态分布
      - 为后续参数/非参数检验提供依据

  【组间差异检验】
      - 独立样本T检验（2组比较）
      - 配对T检验
      - 单因素方差分析ANOVA（3组及以上）
      - Mann-Whitney U检验（非参数）
      - Kruskal-Wallis检验（非参数）
      - 自动进行方差齐性检验（Levene检验）

  【相关性分析】
      - Pearson相关系数（正态分布数据）
      - Spearman秩相关系数（非正态数据）
      - 根据正态性检验结果自动选择

  【回归分析】- 8种方法自动推荐
      (1) 多元线性回归 (OLS)
          - 经典最小二乘法
          - 适用于数据质量良好的情况

      (2) 岭回归 (Ridge)
          - L2正则化
          - 适用于存在多重共线性的数据

      (3) Lasso回归
          - L1正则化
          - 适用于特征筛选和变量压缩

      (4) 弹性网回归 (Elastic Net)
          - L1+L2联合正则化
          - 同时处理共线性和特征筛选

      (5) 稳健回归 (Huber)
          - Huber损失函数
          - 适用于存在异常值的数据

      (6) 分位数回归 (Quantile)
          - 中位数回归
          - 适用于异方差数据

      (7) 虚拟变量回归
          - 自动处理分类变量
          - 将分类变量转换为虚拟变量

      (8) 贝叶斯回归
          - 贝叶斯统计方法
          - 适用于小样本数据

  【模型自动比较】
      - 自动比较OLS、稳健回归、岭回归
      - 根据R²、RMSE等指标选择最优模型
      - 提供模型选择理由说明

  【共线性诊断】
      - VIF (方差膨胀因子)计算
      - 自动检测多重共线性
      - 提供共线性处理建议

5.4 数据清洗模块
--------------------------------------------------------------------------------
  - 自动检测并删除空列
  - 自动检测并删除空行
  - 重复行检测与删除
  - 缺失值智能填充
      * 数值列：均值/中位数填充
      * 分类列：众数填充
  - 异常值检测（IQR方法）
  - 列类型自动识别（数值/分类/日期）
  - 清洗日志完整记录

5.5 可视化模块
--------------------------------------------------------------------------------
  【静态图表】
      - 直方图 (Histogram)
      - 箱线图 (Boxplot)
      - 小提琴图 (Violin Plot)
      - 柱状图 (Bar Chart)
      - 散点图 (Scatter Plot)
      - 相关性热力图 (Correlation Heatmap)

  【交互式图表】
      - Plotly交互式图表
      - 支持缩放、平移、悬停查看
      - 支持图表导出

5.6 AI智能分析模块
--------------------------------------------------------------------------------
  【AI结论生成】
      - 基于分析结果自动生成专业结论
      - 涵盖统计意义和临床意义解读
      - 支持中英文双语

  【AI对话顾问】
      - 交互式问答界面
      - 支持多轮对话
      - 可针对分析结果进行深入探讨
      - 提供数据解读建议

5.7 问题诊断与建议模块
--------------------------------------------------------------------------------
  自动检测以下数据问题并提供改进建议：

  【描述性统计问题】
      - 偏度过大（左偏/右偏）
      - 峰度异常
      - 缺失值比例过高

  【正态性问题】
      - 数据非正态分布
      - 建议数据变换方法

  【T检验问题】
      - 方差不齐
      - 样本量不足
      - 建议非参数检验

  【方差分析问题】
      - 方差不齐
      - 非正态分布
      - 建议Kruskal-Wallis检验

  【相关性问题】
      - 强相关/弱相关解读
      - 多重共线性警告

  【回归问题】
      - 模型拟合度不足
      - 共线性问题
      - 异常值影响
      - 样本量不足

  【数据清洗问题】
      - 大量缺失值
      - 大量重复数据
      - 数据质量问题

5.8 报告生成模块
--------------------------------------------------------------------------------
  【Word报告】
      - 专业学术格式
      - 包含完整分析流程
      - 自动插入图表
      - 图表编号与标注
      - 多语言支持（中文/英文）
      - 可自定义模板类型

  【PDF报告】
      - 基于HTML生成
      - 包含完整分析内容
      - 图表内嵌

  【报告内容结构】
      一、数据清洗摘要
      二、分析方法选择
      三、描述性统计
      四、统计检验结果
      五、回归分析结果
      六、图表
      七、AI分析结论
      八、数据问题与建议
      九、局限性说明

八、API接口说明
================================================================================

【数据上传】
  POST /api/upload
  - 上传CSV/Excel文件
  - 返回：session_id, 列类型, 行列数

【自动分析】
  POST /api/auto-analyze
  - 对当前会话数据进行智能分析
  - 返回：完整分析结果

【生物数据库获取】
  POST /api/bio-database/fetch
  - 从生物医学数据库获取数据
  - 参数：source（数据库源）, identifier（数据标识）
  - 返回：获取的数据信息

【获取分析结果】
  GET /api/results
  - 获取当前会话的分析结果

【获取问题诊断】
  GET /api/issues
  - 获取数据问题诊断与建议

【生成报告】
  GET /api/report
  - 生成Word分析报告
  - 参数：template_type, language, author等
  - 返回：Word文档下载

【健康检查】
  GET /api/health
  - 服务状态检查


九、安装与部署
================================================================================

9.1 本地安装
--------------------------------------------------------------------------------
  (1) 确保Python 3.10+已安装

  (2) 创建虚拟环境：
      python -m venv venv
      venv\Scripts\activate  (Windows)
      source venv/bin/activate  (Linux/Mac)

  (3) 安装依赖：
      pip install -r requirements.txt

  (4) 配置环境变量：
      复制 .env.example 为 .env
      编辑 .env 文件，设置 ARK_API_KEY

  (5) 启动服务：
      python final_app.py
      或双击 start.bat (Windows)

  (6) 访问：http://localhost:12245

9.2 Docker部署
--------------------------------------------------------------------------------
  (1) 构建镜像：
      docker build -t biomed-analytics .

  (2) 运行容器：
      docker run -p 12245:12245 biomed-analytics

  (3) 或使用docker-compose：
      docker-compose up -d

9.3 配置说明
--------------------------------------------------------------------------------
  config.yaml 配置项：
    ai:
      base_url: AI API地址
      model: 模型名称
      max_tokens: 最大token数
      temperature: 温度参数

    server:
      host: 服务地址
      port: 服务端口


十、使用流程
================================================================================

【方式一：自动获取数据】
  1. 点击"数据自动获取"标签
  2. 选择数据库源（如NCBI GEO）
  3. 输入数据标识（如GSE42872）
  4. 点击"获取数据"
  5. 等待数据获取完成
  6. 系统自动进行分析
  7. 查看分析结果和图表
  8. 导出Word报告

【方式二：手动上传数据】
  1. 点击"数据手动获取"标签
  2. 拖拽文件或点击选择文件
  3. 等待文件上传完成
  4. 选择分析模式（智能选择/手动选择）
  5. 点击"开始分析"
  6. 查看分析结果和图表
  7. 导出Word报告

【AI顾问使用】
  1. 点击"AI顾问"标签
  2. 在输入框中输入问题
  3. 获取AI分析建议
  4. 可进行多轮对话
